TIPS DE USO
───────────

🎬 Para padres/stakeholders:
   1. Comienza en Index
   2. Lee las descripciones
   3. Abre Padre → sigue el journey completo
   4. Luego Admin → ve el simulador
   5. Termina en Alumno Secundaria → portfolio

📐 Para diseñadores:
   1. Abre cada prototipo
   2. Abre DevTools (F12) → Modo Responsive
   3. Prueba en mobile (375px), tablet (768px), desktop
   4. Identifica componentes reutilizables
   5. Anota mejoras de UX

💻 Para developers:
   1. Revisa la estructura HTML
   2. Nota los patrones CSS (clases, variables)
   3. Lee el JavaScript (lógica de interacción)
   4. Identifica componentes para reutilizar
   5. Planifica la arquitectura del proyecto real

CÓMO EXTENDER
──────────────

¿Quieres agregar más funcionalidad?

1. Nuevo Tab:
   Copia HTML de un tab existente, actualiza el JavaScript

2. Nuevo Componente:
   Define en styles.css, reutiliza clases existentes

3. Nueva Página:
   Copia un HTML existente, adapta el contenido

4. Animaciones:
   Usa @keyframes en CSS (bounce, slideIn, spin incluidos)
