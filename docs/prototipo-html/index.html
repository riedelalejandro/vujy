💡 Guía de Uso

  1. Elige un perfil del menú superior o de las tarjetas abajo
  2. Lee la descripción del journey que se simula
  3. Interactúa con el chat: escribe mensajes o haz click en botones
  4. Prueba diferentes acciones para ver cómo responde el sistema
  5. Navega entre pantallas usando los botones de acción
  6. Observa los estados de carga, error y confirmación
