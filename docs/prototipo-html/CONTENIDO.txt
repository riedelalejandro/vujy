GUÍA DE FLUJOS PRINCIPALES (Happy Path)
─────────────────────────────────────────

1️⃣  PADRE - Journey P0 (Resumen → Ausencia → Pago)
   INPUT:  "¿Cómo le fue a Mateo esta semana?"
   → Asistente devuelve resumen (95% asistencia, 8.2 promedio, 2 tareas)
   → Input: "Mateo no va a ir mañana"
   → Sistema pide confirmación + motivo (enfermedad)
   → Confirma → Alerta enviada a docente
   → Asistente sugiere "¿Pagamos la deuda?"
   → INPUT: "Sí" → Link de Mercado Pago → Confirma pago
   OUTPUT: Recibo + transacción ID

2️⃣  DOCENTE - Día típico
   1. Abre app → Selectiona clase (4to A)
   2. "Marcar asistencia" → Lista de 20 alumnos → marca ausentes (Sofía)
   3. "Enviar comunicado" → Dicta: "Viernes acto, vengan de blanco"
   4. Asistente genera borrador → Aprueba → Se envía a 25 padres
   5. "Cargar notas" → Tabla de Matemática → Ingresa 4 notas
   6. Sistema alerta: Juan bajó 2 puntos
   7. End of day: "Registrar observación" → Nota sobre Mateo (participativo)

3️⃣  ADMIN - Decisión financiera
   1. Abre dashboard → Ve $47,500 deuda + 23 familias en riesgo
   2. Cliquea "Plan Pago" para Familia Pérez
   3. Va a "Simulador" → Pregunta: "¿Qué pasa si subo cuota 15%?"
   4. Mueve slider → Sistema calcula:
      - Nueva recaudación: $176,500
      - Deserción estimada: -24 alumnos
      - Impacto neto: -$34,000/mes 🔴
   5. Lee recomendación de IA: "Aumentos graduales son mejores"
   6. Decide: aumentar solo 8% + mejorar retención

4️⃣  ALUMNO INICIAL - Sesión de juego
   1. Entra a app → Ve su mascota (leoncito) feliz
   2. "¿Qué actividad quieres?" → Desafío visual: "¿Cuál dice muuuu?"
   3. 3 opciones: 🐄 🦢 🐑
   4. Toca 🐄 → ¡Correcto! Aplausos + 3 estrellitas
   5. Leoncito creció un poco + ganó nivel
   6. Barra de progreso: 2/5 actividades hoy
   7. "¿Una más?" → Siguiente actividad

5️⃣  ALUMNO PRIMARIA - Semana de misiones
   1. Abre app → Ve 450 monedas + Nivel 8
   2. Misiones activas: División (150 💰), Lectura (200 💰), Desafío grupal (500 🏆)
   3. Cliquea "Empezar misión" (División)
   4. Completa 5 problemas → Recibe 150 monedas
   5. "Biblioteca" → Lee "El Quijote" (60% done)
   6. Logro desbloqueado: "Lector Explorador"
   7. Desafío grupal: 5to A va ganando 850/1000

6️⃣  ALUMNO SECUNDARIA - Preparación para prueba
   1. Abre app → Agenda: "Prueba Física jueves"
   2. Cliquea "Hacer simulacro"
   3. Responde 10 preguntas sobre Fuerzas
   4. Resultado: 7.5/10 → Weak areas: Movimiento 3D
   5. Va a "Tutor IA" → "Explicame movimiento en 3D"
   6. Tutor lo guía paso a paso + ejemplos
   7. Va a "Debate" → Participa en foro sobre Revolución Francesa
   8. Profesor comenta: "Buen análisis, expande sobre economía"
   9. Actualiza portfolio → Agrega "Trabajo Historia en progreso"

CONSIDERACIONES UX/UI
─────────────────────

✓ Lenguaje simple: no jerga técnica
✓ Confirmación de acciones críticas (ausencia, pago)
✓ Estados claros: loading, error, empty, success
✓ Feedback inmediato: animaciones, toasts, confirmaciones
✓ Mobile-first: funciona bien en 375px+
✓ Responsive: tablet (768px), desktop (1024px+)
✓ Accesibilidad: buenas prácticas (contraste, tamaño de texto)
✓ Performance: CSS sencillo, JS vanilla (sin dependencias)
